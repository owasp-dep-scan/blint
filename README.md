# Introduction
blint is a simple binary linter to check the security properties and hardcoded credentials in your executables. It is powered by [lief](https://github.com/lief-project/LIEF)

Supported formats:

- ELF
- PE
- Mach-O

## Installation

- Install python 3.8 or 3.9

```bash
pip3 install blint
```
